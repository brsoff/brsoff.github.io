Tableflip

open / github

About: Tableflip continually serves users a information on a random nearby restaurant to help them decide/give them on ideas on where to eat. Users can then get directions instantly from their current location via an embedded Google map via any transit method. In addition, users can favorite restaurants they like, blacklist restaurants and put restaurants on a watchlist to visit later.

Technologies: Ruby on Rails, RSpec, Capybara, jQuery, AJAX, Bootstrap
